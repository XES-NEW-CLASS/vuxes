# Icon 图标

----

提供了一套常用的图标集合

## 如何使用

直接通过设置类名为 `x-icon-iconName` 或者 `tutor-icon-iconName` 来使用即可。例如：

<div class="demo-block">
  <p class="icon-box">
    <i class="tutor-icon-ArrowRight"></i>
    <i class="x-icon-delete"></i>
  </p>
</div>

::: demo
```html
<i class="tutor-icon-ArrowRight"></i>
<i class="x-icon-delete"></i>
```
:::
