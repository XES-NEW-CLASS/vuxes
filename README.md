`vuxes` 是一款基于 `Vue.js 2.0` 的前端 UI 组件库，主要用于学而思新课堂项目

## 特性

- 基于 `Vue` 开发的 UI 组件
- 使用 Vue Cli3 扩展构建，支持 ES2015
- 提供友好的 API，可灵活的使用组件

## 安装

- yarn

```bash
yarn add vuxes
```

- npm

```bash
npm install vuxes -S
```

## 使用

```js
import vuxes from 'vuxes' // 引入组件库
import 'vuxes/lib/theme-default/index.css' // 引入样式库

Vue.use(vuxes)
```
